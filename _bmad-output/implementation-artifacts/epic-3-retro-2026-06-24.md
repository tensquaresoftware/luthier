## Successes

1. **Incremental migration** ‚Äî pytest tier added alongside legacy `test_story_*.py` unittest modules without breaking existing regression.
2. **AD-8 compliance** ‚Äî Unit and integration tests import `core/*` only; no Qt side effects in the default test path.
3. **Round-trip fidelity** ‚Äî Integration tier verifies `ProjectSpec ‚Üí generate ‚Üí read` on all fields, CMake fallback complete-or-`None` guard, and AC4 `CMakeLists.txt` hygiene.
4. **Shared helpers** ‚Äî `tests/conftest.py` centralizes `make_spec`, tree comparison, and writer/generator helpers deferred since 3-1.
5. **Code review discipline** ‚Äî Each story passed adversarial review; findings triaged into patches, deferrals, or dismissals.

---

## Challenges & Growth Areas

1. **Duplication with legacy unittest** ‚Äî `test_story_2_1.py` / `test_story_2_2.py` overlap pytest integration coverage; intentional until a future cleanup story.
2. **NFR1 line limits** ‚Äî Several helpers/tests exceeded 15-line guidance; resolved in 3-4 review via extraction to `conftest.py`.
3. **Deferred hardening** ‚Äî Edge cases (parametrize `plugin_type`, malformed sidecar, broader CMake regex variants, subprocess no-Qt guard) documented in `deferred-work.md`.
4. **Coverage gaps from earlier epics** ‚Äî `juce_dir` pipeline E2E, gitignore override UI flows, and post-render JSON validation remain untested (scoped out of Epic 3 AC).

---

## Key Learnings

1. **Defer shared fixtures explicitly** ‚Äî Stories 3-1/3-3 deferred `conftest.py`; 3-4 delivered it cleanly once integration tests needed it.
2. **Test production contracts, not implementation details** ‚Äî Integration tests assert file presence, field equality, and byte-for-byte regenerate fidelity without coupling to internal writer methods beyond AC requirements.
3. **Helper consistency matters** ‚Äî `write_project` must mirror `ProjectGenerator` path resolution (`Path(spec.destination_dir) / project_name`); divergent helpers hide bugs.
4. **Review layers catch different issues** ‚Äî Acceptance Auditor confirmed AC compliance; Blind Hunter and Edge Case Hunter surfaced quality hardening beyond AC.
